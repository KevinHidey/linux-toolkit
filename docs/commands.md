# Used Linux Commands

This project uses several basic Linux commands.

## General commands

- `hostname` shows the computer name.
- `uname -r` shows the kernel version.
- `uptime` shows how long the system has been running.
- `df -h` shows disk usage.
- `free -h` shows memory usage.
- `ip addr` shows network addresses.
- `ip route` shows routing and gateway information.
- `tar -czf` creates compressed backups.
- `tar -xzf` extracts compressed backups.
- `awk` processes text output.
- `wc -l` counts lines.
- `chmod +x` makes scripts executable.
- `git add`, `git commit`, and `git push` manage project changes.
